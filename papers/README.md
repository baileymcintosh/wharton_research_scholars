# Paper Library

This folder is organized by research theme:

- `costly_information`: information acquisition, equilibrium information production, and informational efficiency.
- `value_of_information`: decision-theoretic work on valuing information.
- `prediction_asset_pricing`: predictive modeling, fundamentals forecasting, and statistical signal extraction in finance.
- `alternative_data`: text, media, big-data, and alternative-dataset papers relevant to monetizable informational edge.
- `crowding_alpha_decay`: arbitrage limits, capacity, crowding, signal decay, and equilibrium erosion of alpha.
- `metadata`: machine-readable catalog with citation metadata, abstracts, links, and local file paths for downloaded PDFs.

The catalog files are:

- `papers/metadata/paper_catalog.json`
- `papers/metadata/paper_catalog.csv`

Open-access PDFs are downloaded when reachable directly. When a paper is not openly downloadable, the metadata catalog still records the DOI and landing page so the paper remains part of the organized literature set.
